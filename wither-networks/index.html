Frequently Asked Questions

How do I connect to the server?

Simply copy our server IP (play.withernetworks.fun) and paste it into the Minecraft server address field. No additional mods or resource packs are required.

What versions of Minecraft are supported?

We support Minecraft Java Edition versions 1.16.5 and above. Our server is regularly updated to support the latest Minecraft versions.

Is the server free to play?

Yes, our server is completely free to play. We offer optional cosmetic ranks and items through our store to support server development.

How can I report a player or issue?

You can report issues through our in-game reporting system, forums, or by contacting our support team directly through Discord.
